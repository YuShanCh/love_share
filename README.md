# love_share
使用套件:  
- java
- spring boot 
- HTML
- CSS
- JavaScript
- Bootstrap  
# 專案介紹:  
-   1. 基金會透過公開資料使民眾可得知基金會所需物資類別、詳細數量。
-   2. 讓捐贈者自己認領數量將適合的物資捐贈到需要的單位。
-   3. 解決基金會擔心收到過多且不必要的物資。
